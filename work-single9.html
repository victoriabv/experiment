Tasca 9

Càlcul de les quotes d'una hipoteca

(clica aquí o a la imatge de l'esquerra per accedir a la pràctica)
Fes una pàgina web que ens mostri un formulari per poder cercar fotografies a Flickr, utilitzant Bootstrap i jQuery.

El formulari ens ha de demanar dos camps, tots ells requerits:

Etiquetes. En podem posar més d'una, separades per espais Tipus de cerca. Serà un desplegable que ens permetrà escollir si volem cercar "Totes" les etiquetes o "Qualcunes" d'elles. Per defecte, estarà seleccionat "Totes" La cerca s'ha de fer mitjançant el mètode getJSON de jQuery, i es farà sobre la URL "https://api.flickr.com/services/feeds/photos_public.gne", agafant els paràmetres tags i tagmode del formulari anterior. Fixau-vos que al paràmetre tags les etiquetes han d'anar separades per comes, i al camp "etiquetes" del formulari les teniu separades per espais (haureu de fer la conversió). També haurem d'especificar el paràmetre format=json, i el paràmetre jsoncallback=?. Podeu trobar ajuda de com utilitzar aquesta API a https://www.flickr.com/services/feeds/docs/photos_public/ i a https://www.flickr.com/services/api/response.json.html.

Quan la cridada AJAX ens retorni el resultat en format JSON, ho haurem de dibuixar en una taula de Bootstrap, mostrant la següent informació:

Títol de la foto (title) Enllaç cap a la pàgina de Flickr (link) Miniatura de la foto (media) Data de publicació (published) Autor (author) Etiquetes (tags) Cada pic que es faci una consulta, la taula de resultats s'ha de redibuixar completament.

Els fitxers han d'estar a un repositori de GitHub.com. Una vegada creat el repositori pots baixar-lo (clone) al teu disc per fer-hi feina al teu ordinador. > git clone

Has de lliurar tots els fitxers necessaris per instal·lar aquesta web a qualsevol servidor web (fitxers html, imatges, etc....), mitjançant un enllaç a una release (publicació) d’aquest repositori de GitHub.

How to create releases: https://help.github.com/articles/creating-releases/" NOTA: als fitxers HTML, CSS, JS s'han de fer tots els comentaris necessaris, com el nom de l'autor i totes les anotacions que facin més comprensible el codi font.
