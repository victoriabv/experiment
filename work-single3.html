Tasca 3

Web d'una agència de viatges

(clica aquí o a la imatge de l'esquerra per accedir a la pràctica)
Desenvolupar la web d'una agència de viatges, seguint un disseny parescut al fet per la web eDreams (http://www.edreams.es/). La web ha de tenir els següents requisits:

-La web de l'agència de viatges ha d'estar a una subcarpeta del teu servidor web. Per exemple, a localhost/tasca3
-Ha de tenir una pàgina, la principal (index.html)
-A la part superior hi ha d'haver una capçalera amb el logo de l'agència i un menú d'accés a totes les activitats ofertades (vols, hotels, lloguer de cotxes, etc...)
-A la part inferior hi ha d'haver un menú d'accés a totes les activitats ofertades (igual que al menú principal) i també el copyright
-Hi has d'incloure algunes fotografies, text i preu de les principals ofertes (mínim 4)
-Ha d'incloure un formulari de cerca d'hotels, on s'hi ha de poder escollir com a mínim el nom de l'hotel, població, un desplegable per número d'estrelles, data d'entrada i de sortida, número de persones i si es volen habitacions individuals o no.
-El formulari ens ha d'enviar a la mateixa pàgina principal (index.html).
-Els enllaços que no s'hagi especidificat el seu destí poden dur a la mateixa pàgina principal o a cap pàgina (posant com a href "#" o "javascript:;")
-L'aspecte s'ha de controlar mitjançant un full d'estils CSS extern, amb un fitxer que enllaçarem a la capçalera de la pàgina HTML.
-IMPORTANT: no es poden utilitzar taules. En el seu lloc, utilitzar elements "div".
-Has de lliurar un zip amb tots els fitxers necessaris per instal·lar aquesta agència de viatges a qualsevol servidor web (fitxers html, css, imatges, etc....).

NOTA: a les pàgines HTML i els fulls d'estils CSS s'han de fer tots els comentaris necessaris, com el nom de l'autor i totes les anotacions que facin més comprensible el codi font de la pàgina.

OPCIONAL: penja aquesta web a un servidor web online (Codeanywhere). Per fer-ho, hauràs de crear un compte gratuït en aquesta web, i després crear un contenidor de tipus HTML a damunt un Ubuntu 14.04 (File -> New Connection -> Container -> HTML - Ubuntu 14.04). Després, hauràs de pujar la teva web a la carpeta principal del servidor. Una vegada pujada, la web serà accessible des de la URL http://
